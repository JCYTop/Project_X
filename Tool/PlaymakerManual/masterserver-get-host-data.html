MasterServer Get Host Data

Get host data from the master server.

Host Index
The index into the MasterServer Host List
Use Nat
Does this server require NAT punchthrough?
Game Type
The type of the game (e.g., 'MyUniqueGameType')
Game Name
The name of the game (e.g., 'John Does's Game')
Connected Players
Currently connected players
Player Limit
Maximum players limit
Ip Address
Server IP address.
Port
Server port
Password Protected
Does the server require a password?
Comment
A miscellaneous comment (can hold data)
Guid
The GUID of the host, needed when connecting with NAT punchthrough.

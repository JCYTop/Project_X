Error Checker

The Error Checker finds setup errors as you edit FSMs. This allows you to catch and fix errors before you even hit play.

Click on the error message in the Error Check Window to jump directly to the error.

The Error Checker shows errors at different levels in the UI:

1. An Error Count in the Debugging Toolbar:

Clicking on the Error Count opens the Error Check Window.

2. Error Icons in the Graph View, Fsm Browser, Template Browser and State Browser.

This lets you see at a high level that some FSMs have errors and then drill down from there.

3. Error Boxes in Action Editors.

NOTE: Some error boxes are clickable, letting you automatically fix the error!

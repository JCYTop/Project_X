Look At 2d

Rotates a 2d Game Object on it's z axis so its forward vector points at a 2d or 3d position.

Game Object
The GameObject to rotate.
Vector 2 Target
The 2d position to Look At.
Vector 3 Target
The 3d position to Look At. If not set to none, will be added to the 2d target
Rotation Offset
Set the GameObject starting offset. In degrees. 0 if your object is facing right, 180 if facing left etc...
Debug
Draw a debug line from the GameObject to the Target.
Debug Line Color
Color to use for the debug line.
Every Frame
Repeat every frame.

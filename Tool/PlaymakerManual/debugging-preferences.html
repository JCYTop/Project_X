Debugging Preferences

Debugging Preferences control the debugging in Playmaker when the game is running in the editor.

Show State Labels in Game View

Show the current state of FSMs in the Game View. NOTE: Requires a PlayMakerGUI component in the scene.

Enable DebugFlow

Debug Flow lets you pause the game and step back through state changes.

Enable Transition Effect

Toggles the glow effect on activated transitions.

Jump to Breakpoint/Error

Automatically switch to the FSM/State that caused a break or error.

Forward PlaymakerLog to Unity Log

Forward  Playmaker log messages to the Unity Console log. The Playmaker Log helps you sort events by FSM and quickly follow Events, whereas the Unity Console lets you see all log messages in a single list. Both views can be useful when debugging.

Debug Look At Color

Look At actions can draw a debug ray in the Scene View. This controls the color of that ray.

Debug Raypick Color

Raycast actions can draw a debug ray in the Scene View. This controls the color of that ray.

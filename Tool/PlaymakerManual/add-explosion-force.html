Add Explosion Force

Applies a force to a game object that simulates explosion effects. The explosion force will fall off linearly with distance. Use the Explosion action to apply explosion force to all objects in a blast radius.

Game Object
The game object to apply the force to.
Center
The center of the explosion.
Force
The force of the explosion.
Radius
The radius of the explosion. Force falls off linearly with distance.
Upwards Modifier
Applies the force as if it was applied from beneath the object. This is useful since explosions that throw things up instead of pushing things to the side look cooler. A value of 2 will apply a force as if it is applied from 2 meters below while not changing the actual explosion position.
Force Mode
The type of force to apply.
Every Frame
Do every frame while the state is active.
